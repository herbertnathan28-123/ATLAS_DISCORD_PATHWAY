# Dark Horse FOH v6 — Brief #8 Language Audit

Status: FOLLOW-UP ISSUE UPDATED WITH BRIEF #8

Brief #8 is a global wording, terminology, hyperlink, and chart-language rule for Dark Horse FOH v6.

## Rule

Dark Horse output must avoid rough / unclear / semi-retail phrases and explain every important trading term in plain language without reducing depth.

## Removed from user-facing Dark Horse output

- prints
- Trigger Level / trigger
- broken level
- floor / ceiling metaphors
- reclaim / reclaimed
- fighting structure
- better setup / cleaner setup
- give the trade more room
- either side
- marginal setup
- late-stage caveat
- path of least resistance
- confirmed directional structure
- structural anchors

## Required replacement language

- Trigger Level -> Decision Level
- "prints a close" -> "forms/closes/confirms with a candle close"
- floor/ceiling metaphors -> "price closed above/below the decision level" and "buyers/sellers defended that area"
- "better/cleaner setup" -> explicit quality criteria: stronger conviction, price closer to entry, better reward-to-risk, less invalidation pressure
- vague structure references -> named levels: decision level, entry zone, watch level, invalidation

## Chart alignment

Rendered chart labels must use the same language as the card:

- DECISION LEVEL
- ENTRY ZONE
- WATCH LEVEL
- INVALIDATION
- BREAK ABOVE / BREAK BELOW
- RETEST HELD
- FAILED RECOVERY
- BUYERS DEFENDING / SELLERS DEFENDING

The chart-card remains a proof-of-idea teaching visual, not decorative output.
